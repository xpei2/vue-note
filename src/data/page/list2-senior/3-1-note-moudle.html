• 定义：let value1 = 10, value2 = 99, let value3 = 52;
  • 导出对象：export{value1, value2, value3};
  • 导出值：export let num = 200;
  • 导出函数：export function mul(num, value1) { return num + value1; }
  • 导出类：export class Person { run() { console.log('导出了一个类'); } }class ,es6类的语法
  • 导入对象1：import { value1 as a, value2 as b, value3 as c } from './js/export.js';as：给导入数据重命名
  • 导入对象2：import { value1, value2, value3 } from './js/export.js';
  • 导入值：import { num } from './js/export.js';
  • 导入函数：import { mul } from './js/export.js';
  • 导入类：import { Person } from './js/export.js';
  • 通配符方法：import * as 自定义对象名 from './js/export.js';就是将导出的所有东西作为一个对象导入
  • 导出默认：export default {}，此方法导入的时候不需要加花括号
